Question:

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing scripts like JavaScript to access, manipulate, and update the content, structure, and style of a webpage dynamically.

2. How do you select an element from the DOM?

document.getElementById('id') selects an element by its ID. document.querySelector('selector') selects the first element that matches a CSS selector. document.getElementsByClassName('class') selects elements by class name.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events triggered by its child elements. This is useful because it improves performance by reducing the number of event listeners and handles dynamic elements added to the DOM after the listener is set.

4. How do you manipulate an element's attributes and styles using the DOM?

Attributes: Use element.setAttribute('attr', 'value') to set an attribute, and element.getAttribute('attr') to get it. Styles: Use element.style.property = 'value' to modify inline styles, or add/remove CSS classes with element.classList.add('class') and element.classList.remove('class').
